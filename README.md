### 사용법
* ```git clone https://github.com/nnnlog/codeup_AC_download```
* ```cd codeup_AC_download/```
* ```npm install```
* ```auth.js``` 를 열고 본인의 id와 비밀번호 입력
* ```node index.js```
---
```
/codeup_AC_download/code/ 에 풀이코드가 다운로드됨.

이 프로그램으로 발생하는 모든 문제는 사용자 본인이 책임집니다.
```
